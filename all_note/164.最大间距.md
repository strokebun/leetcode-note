### 164.最大间距



**思路1：** 直接排序

``` java
class Solution {
    public int maximumGap(int[] nums) {
        Arrays.sort(nums);
        int maxGap = 0;
        for (int i = 1, len = nums.length; i < len; i++) {
            maxGap = Math.max(maxGap, nums[i] - nums[i-1]);
        }
        return maxGap;
    }
}
```

**思路2：** 基数排序

题目要求 O(n) 的时间和空间复杂度，符合要求的有基数排序

``` java
class Solution {
    public int maximumGap(int[] nums) {
        int n = nums.length;
        if (n < 2) {
            return 0;
        }
        long exp = 1;
        int[] buf = new int[n];
        int maxVal = Arrays.stream(nums).max().getAsInt();

        while (maxVal >= exp) {
            int[] cnt = new int[10];
            for (int i = 0; i < n; i++) {
                int digit = (nums[i] / (int) exp) % 10;
                cnt[digit]++;
            }
            for (int i = 1; i < 10; i++) {
                cnt[i] += cnt[i - 1];
            }
            for (int i = n - 1; i >= 0; i--) {
                int digit = (nums[i] / (int) exp) % 10;
                buf[cnt[digit] - 1] = nums[i];
                cnt[digit]--;
            }
            System.arraycopy(buf, 0, nums, 0, n);
            exp *= 10;
        }

        int ret = 0;
        for (int i = 1; i < n; i++) {
            ret = Math.max(ret, nums[i] - nums[i - 1]);
        }
        return ret;
    }
}
```

**思路3：**基于桶的算法

数组长度为 n，数组的最大值、最小值分别为 max、min，可以看出最大间隔不小于 `d = (max-min)/n`，整个区间划分为若干个大小为 `d` 的桶，并将数组元素放到对应桶中。**最大间隔一定是一个桶的最小值和前一个桶的最大值之差**（去除空桶）。

``` java
class Solution {
    public int maximumGap(int[] nums) {
        int n = nums.length;
        if (n < 2) {
            return 0;
        }
        int min = Arrays.stream(nums).min().getAsInt();
        int max = Arrays.stream(nums).max().getAsInt();
        int d = Math.max(1, (max - min) / (n - 1));
        int bucketSize = (max - min) / d + 1;
        int[][] bucket = new int[bucketSize][2];
        // 设置初始值为MIN_VALUE
        for (int i = 0; i < bucketSize; i++) {
            bucket[i][0] = Integer.MIN_VALUE;
        }

        for (int num : nums) {
            int idx = (num - min) / d;
            // 空桶
            if (bucket[idx][0] == Integer.MIN_VALUE) {
                bucket[idx][0] = bucket[idx][1] = num;
            } else {
                bucket[idx][0] = Math.min(num, bucket[idx][0]);
                bucket[idx][1] = Math.max(num, bucket[idx][1]);
            }
        }

        int maxGap = 0;
        int prev = -1;
        for (int i = 0; i < bucketSize; i++) {
            // 跳过空桶
            if (bucket[i][0] == Integer.MIN_VALUE) {
                continue;
            }
            if (prev != -1) {
                maxGap = Math.max(maxGap, bucket[i][0] - bucket[prev][1]);
            }
            prev = i;
        }
        return maxGap;


    }
}
```
